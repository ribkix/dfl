Server Rules

If you don't follow the following rules, your uploaded server will be declined.

  • The uploaded server must be owned by you.
  • The uploaded server must abide with the Discord Terms of Service.
  • The uploaded server must not include spam in the description.
  • The uploaded server must not include any NSFW content (includes the icon and the description).
  • You must be in our server (https://discord.gg/9U8PMjC).
  • If you chose to use your Discord as contact, make sure we are able to DM you.

Extra notes:

  • If your server is not checked within the 3 days you will contact you with the reason why it's taking longer than usual.
  • We have the right to decline your server without the reason being listed in our rules.
